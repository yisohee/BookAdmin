## 🛠️ 사용 방법

### 0. 구동 방법

 - 명령 프롬프트(cmd) 실행
 -  해당 JAR 파일이 있는 디렉터리로 이동  
 - 아래 명령어로 프로그램 실행:
   ```bash
   java -jar [jar파일 이름].jar
